# Student QA (SQA) Checklist

**Goal**: Verify the course is student-ready by checking navigation, content visibility, and link validity from the **Student Preview** perspective.

> **Related**: See the [SQA Advice & Preparation Guide](sqa_advice.md) for pro-tips and common pitfalls.

---

## 1. Course Info

- [ ] **1. Course Code** — Enter the course code.
- [ ] **2. Status Change** — Change tracking list status to **"Under SQA"**.

---

## 2. Orientation & Modules

- [ ] **3. Module Visibility** — All required modules are visible to students (Start Here, Student Resources, Orientation, Modules 1–15).
- [ ] **4. Student/Program Resources** — "Student Resources" (PBSN) / "Program Resources" (A&S) module is visible and contains relevant content.
- [ ] **5. Start Here Instructions** — "Start Here" module provides clear student instructions for getting started.
- [ ] **6. Course Outline (MSN)** — For MSN courses, a course outline and schedule document is provided *(N/A for non-MSN)*.

---

## 3. Syllabus & Banner

- [ ] **7. Banner Image** — Banner image matches the course code and title.
- [ ] **8. Syllabus Resource** — "Course Overview and Syllabus" resource is present in the Start Course Orientation module; link is valid and opens correctly.
- [ ] **9. Header Info (TDAC)** — Header contains the correct class code and name *(TDAC courses only)*.
- [ ] **10. Syllabus Links** — All links within the syllabus work correctly.
- [ ] **11. Comments/Revisions** — Comments and revision marks are hidden.
- [ ] **12. Track Changes Colors** — No colors reflecting tracked changes are visible.
- [ ] **13. Syllabus Format** — Syllabus has a clean, professional format with no layout issues.
- [ ] **14. TDAC Resources** — "Syllabus part 2" and "Setting up for success" resources are visible *(TDAC only; N/A for others)*.
- [ ] **15. Orientation Acknowledgement** — "Course Orientation Acknowledgement" test completion correctly releases all modules.

---

## 4. Links

- [ ] **16. Module Release Conditions** — All module release conditions are correctly set up.
- [ ] **17. Raw Links** — No raw links in any Blackboard page *(ATI resources excepted)*.
- [ ] **18. Learning Materials Links** — All links in the "Learning Materials" module work, including video links and LTI links.

---

## 5. Assignments

- [ ] **19. Has Assignments** — Course has assignments.
- **If Yes:**
    - [ ] **20. Assignment Banner** — Banner matches the activity type.
    - [ ] **21. Assignment Instructions** — Instructions have clear guidelines for students.
    - [ ] **22. Assignment Links/Attachments** — All links and attachments work and open in a new tab; no raw links.
    - [ ] **23. Assignment Rubrics** — All rubrics are visible to students and match SharePoint files.

---

## 6. Discussions

- [ ] **24. Has Discussions** — Course has discussions.
- **If Yes:**
    - [ ] **25. Discussion Banner** — Banner matches the activity type.
    - [ ] **26. Discussion Instructions** — Instructions have clear guidelines for students.
    - [ ] **27. Discussion Links/Attachments** — All links and attachments work and open in a new tab; no raw links.
    - [ ] **28. Discussion Rubrics** — All rubrics are visible to students and match SharePoint files.

---

## 7. Tests

- [ ] **29. Has Tests** — Course has tests.
- **If Yes:**
    - [ ] **30. Test Banner** — Banner matches the activity type.
    - [ ] **31. Test First Page** — Information is correctly displayed on the first page.
    - [ ] **32. Test Guidelines** — Guidelines match the panel requirements.

---

## 8. Journals

- [ ] **33. Has Journals** — Course has journals.
- **If Yes:**
    - [ ] **34. Journal Banner** — Banner matches the activity type.
    - [ ] **35. Journal Instructions** — Instructions have clear guidelines for students.
    - [ ] **36. Journal Links/Attachments** — All links and attachments work and open in a new tab; no raw links.
    - [ ] **37. Journal Rubrics** — All rubrics are visible to students and match SharePoint files.

---

## 9. Blackboard Tabs

- [ ] **38. Discussions Tab Order** — Discussions follow a sequential order (e.g., "Introductions" → "Module 1" → "Module 2" … "Module 15").
- [ ] **39. Gradebook Order** — Gradebook items are organized chronologically or sequentially (Module 1 → Module 15).
- [ ] **40. Groups Tab** — Self-enrollment groups are set up correctly *(if applicable)*.

---

## 10. Wrap-Up

- [ ] **41. Add Findings** — Add findings to the Microsoft tracking list.
- [ ] **42. Time Log** — Add time spent on QA as a comment in the SharePoint list *(e.g., "QA time: 15 mins")*.
- [ ] **43. Status Change** — Change tracking list status to **"SQA findings"**.
