# Teacher QA (TQA) Checklist

**Goal**: Verify the course is instructor-ready by performing a comprehensive quality check across all content areas, assessments, and gradebook configuration.

> **Related**: See the [TQA Advice & Preparation Guide](tqa_advice.md) for pro-tips and common pitfalls.

---

## 1. Introduction

- [ ] **1. Course Code** — Enter the course code.
- [ ] **2. Status Change** — Change tracking list status to **"Under TQA"**.

---

## 2. Progress Tracking

- [ ] **3. Progress Tracking** — Progress tracking is enabled for the course.

---

## 3. Course Content & Staple Items

- [ ] **4. Syllabus Match** — Syllabus matches SharePoint and Blackboard content.
- [ ] **5. Course Key Verbiage** — Course Key verbiage is placed inside the "Start Here" module *(TDAC & Med CAN only)*.
- [ ] **6. Course Template** — Course template is up to date.
- [ ] **7. LTIs Present** — Correct LTIs are present per the SharePoint list.
- [ ] **8. JBL Links** — All JBL links open in a new window and redirect to the correct resource.

---

## 4. Course Outline (Module by Module)

- [ ] **9. Release Conditions** — Release conditions allow student access after Course Orientation Acknowledgment *(not TDAC)*.
- [ ] **10. Module Documents** — Module documents match Syllabus/SharePoint files.
- [ ] **11. No Raw Links** — No raw links; all links open in a separate tab *(ATI & references excepted)*.
- [ ] **12. Reading Materials** — Reading Materials follow APA 7 guidelines.
- [ ] **13. Videos** — All videos show title, duration, CC availability, and a full reference.
- [ ] **14. SCORM Packages** — All SCORM packages work and open in a separate window.

---

## 5. Due Dates

- [ ] **15. Due Dates** — Due dates are adjusted for all assessments.

---

## 6. Assignments

- [ ] **16. Has Assignments** — Course has assignments.
- [ ] **17. Assignment Content** — Assignment content matches Syllabus/SharePoint (title, instructions, attachments).
- [ ] **18. Assignment Links** — All links work and open in a separate tab; no raw links.
- [ ] **19. Assignment Settings** — Assignment settings match program requirements.
- [ ] **20. Assignment Rubrics** — All rubrics match the Syllabus/SharePoint files.
- [ ] **21. Rubric Visibility** — Rubrics are not visible in assignment instructions; no attachment/link leads to rubric.

---

## 7. Discussions

- [ ] **22. Has Discussions** — Course has discussions.
- [ ] **23. Discussion Content** — Discussion content matches Syllabus/SharePoint (title, instructions, attachments).
- [ ] **24. Discussion Links** — All links work and open in a separate tab; no raw links.
- [ ] **25. Discussion Settings** — Discussion settings match program requirements.
- [ ] **26. Discussion Rubrics** — All rubrics match the Syllabus/SharePoint files.
- [ ] **27. Rubric Visibility** — Rubrics are not visible in discussion instructions; no attachment/link leads to rubric.
- [ ] **28. Discussion Tab** — Discussion tab matches discussions from the content tab.

---

## 8. Tests

- [ ] **29. Has Tests** — Course has tests.
- [ ] **30. Test Instructions** — Test instructions match SharePoint/Syllabus and are on a single page.
- [ ] **31. Test Questions & Answers** — Test questions and answers match SharePoint files.
- [ ] **32. Points per Question** — All tests have correct points per question.
- [ ] **33. Question Types** — All questions have the correct type (Matching, T/F, MC, Essay, etc.).
- [ ] **34. Answer Formatting** — Answers are formatted properly (no bold/highlighted options).
- [ ] **35. Test Release Conditions** — Test release conditions match Syllabus/SharePoint instructions.
- [ ] **36. Test Settings** — Test settings match program requirements.
- [ ] **37. Respondus Settings** — Respondus settings match program requirements *(use personal account)*.

---

## 9. Journals

- [ ] **38. Has Journals** — Course has journals.

---

## 10. Gradebook

- [ ] **39. Final Grade Item** — "Final Grade" item is present with correct settings *(year set to far future)*.
- [ ] **40. Gradebook Settings** — Gradebook settings match program requirements.
- [ ] **41. Grading Schema** — "Pacific College Letter" grading schema matches required parameters.
- [ ] **42. Overall Grade** — "Overall Grade" grade item exists.
- [ ] **43. Grade Distribution** — Overall grade distribution matches syllabus; course total = 100%.
- [ ] **44. Gradebook Sorting** — All Gradebook items sorted correctly *(Final Grade → Overall Grade → COA → module activities in order)*.

---

## 11. Choice Polls / Self-Enrollment Groups

- [ ] **45. Choice Polls** — Course has choice polls.

---

## 12. Wrap-Up

- [ ] **46. Add Findings** — Add findings to Microsoft tracking list.
- [ ] **47. Time Log** — Add time spent on TQA as a comment in the SharePoint list *(e.g., "TQA time: 1h 40m")*.
- [ ] **48. Status Change** — Change tracking list status to **"TQA findings"**.
